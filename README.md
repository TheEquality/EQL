# The Equality — EQL Token

**The world's first charity cryptocurrency.**
4% of every sell transaction goes directly to families in economic difficulty — automatically, on-chain, verifiable.

---

## Overview

The Equality (EQL) is a BEP-20 token on the BNB Smart Chain built to create real, measurable social impact through blockchain technology. Every sell transaction triggers an automatic 4% transfer to the charity wallet, which is then distributed to ISEE-verified families as stablecoin payments via VISA card — no intermediaries, no delays.

A further 1% of every transaction is permanently burned, making EQL deflationary by design.

## Key Features

- **Automatic charity distribution** — 4% of every sell is routed on-chain to verified families in need, with no manual intervention required.
- **Deflationary mechanics** — 1% of every transaction is burned permanently, reducing circulating supply over time.
- **Full on-chain transparency** — all Safe multisig wallets, treasury allocations and token locks are publicly verifiable on BscScan.
- **First Came, First Served program** — families can apply directly through the platform with ISEE documentation to join the whitelist and receive monthly support.
- **Multilingual** — the platform and Whitepaper 3.0 are available in 10 languages (IT, EN, ES, FR, DE, PT, ZH, JA, AR, RU).
- **CertiK Audited** — smart contract security reviewed and certified by CertiK.

## Charity Program

Whitelisted families receive up to **€1,000/month** via VISA card. Eligibility is verified through the ISEE (Italian economic hardship certification) process. The charity wallet address is public and all distributions are auditable on-chain.
