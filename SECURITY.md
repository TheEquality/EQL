# Security Policy

## Supported Versions

Only the currently deployed version of The Equality platform and smart contract is actively maintained and eligible for security disclosures.

| Component | Status |
|---|---|
| EQL Smart Contract (BSC) | Active |
| theequality.io Platform | Active |

---

## Reporting a Vulnerability

**Please do not report security vulnerabilities through public GitHub issues.**

If you discover a vulnerability — in the smart contract, the platform, or any associated infrastructure — report it privately via email:

**security@theequality.io**

Include in your report:
- A clear description of the vulnerability
- Steps to reproduce or a proof of concept
- The potential impact (funds at risk, user data, platform integrity)
- Your contact information (optional, for coordinated disclosure)

|---|---|

---

## Responsible Disclosure

We ask that you:
- Give us reasonable time to investigate and address the issue before any public disclosure
- Avoid accessing, modifying, or deleting data that does not belong to you
- Act in good faith toward our users and the project

We commit to:
- Acknowledging your report promptly
- Keeping you informed of progress
- Crediting responsible disclosures publicly (if desired)
